# UART Transmitter in Verilog HDL

A modular UART (Universal Asynchronous Receiver/Transmitter) transmitter designed in Verilog HDL. This project implements serial data transmission using a finite state machine (FSM), baud rate generator, shift register, and bit counter. The design is fully synthesizable and verified through simulation.

---

## Features

- Modular RTL design
- FSM-based transmitter control
- Configurable baud rate generator
- 8-bit UART transmission
- Start and stop bit generation
- Synthesizable Verilog HDL
- Comprehensive testbench for verification
